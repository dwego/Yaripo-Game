# Yaripo

**Yaripo** is a short 2D Metroidvania set in the heart of the Amazon rainforest. You play as **Aruana**, a young woman sacrificed to the abyss of Misty Peak (*Pico da Neblina*) due to the corruption and lies of her own father, the village Shaman.

Surviving the fall into a mystical lake, Aruana must explore the depths of the mountain, reclaim tools left behind by illegal miners, and uncover the dark truth about the pollution of the Black River and her family's betrayal.

## Features

- **Challenging Gameplay:** High-precision platforming and exploration.
- **Unique Tools:** 
    - **Grappling Hook:** Swing through cavern ceilings.
    - **Pickaxe:** Break through cracked walls to find secrets.
    - **Boots:** Walk safely through toxic acid waste.
- **Deep Narrative:** Discover the conspiracy between the Shaman and illegal miners.
- **Atmospheric Visuals:** Explore hand-drawn sprites of Aruana and the mystical entities of the Amazon.

## Compatibility

Yaripo is built using Java, which means it is **cross-platform** and runs on:
- **Windows**
- **macOS**
- **Linux**

(Requires Java 17 or higher)

## How to Play

### Download the Latest Release
You can download the game's executable JAR file from our releases page:

👉 **[Download Yaripo Release](https://github.com/user/YaripoGame/releases/latest)**

### Running the Game
1. Ensure you have **Java 17** (or later) installed on your system.
2. Download the `yaripo.jar` file.
3. Run it by double-clicking the file or using the terminal:
   ```bash
   java -jar yaripo.jar
   ```

## Development

If you wish to compile the project from source:
1. Clone the repository.
2. Run Maven to build:
   ```bash
   mvn clean package
   ```
3. The executable will be generated in the `target/` folder as `yaripo.jar`.
